# Samples

These `.dfont` files are included so visitors can try Glitch.dfont without creating a source file first.

Included samples:

- `facade-ouest.dfont` - Facade Ouest by Eleonore Fines, Velvetyne Type Foundry; licensed under the SIL Open Font License 1.1; source: https://velvetyne.fr/fonts/facade/
- `trickster-regular.dfont` - Trickster by Jean-Baptiste Morizot; licensed under the SIL Open Font License 1.1; source/process: https://medium.com/@jbmorizot/trikster-a-postmortem-686f4608da9e
- `picnic-regular.dfont` - PicNic by Mariel Nils; licensed under CUTE 0.1; source: https://velvetyne.fr/fonts/picnic/

License and source-context files are bundled under `licenses/`:

- `licenses/facade/`
- `licenses/trickster/`
- `licenses/picnic/`

PicNic's CUTE license asks redistributors to credit the designer, provide source context, and keep the accompanying files with the font. For that reason, the PicNic sample includes its license text, CUTE PDF, README, copyright, font log, trademark notice, and `.glyphs` source file.
