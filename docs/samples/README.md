# Samples

With these `.dfont` files, anyone can try **Glitch.dfont** without creating a source file first.

Currently, the app doesn't support downloading a glitched sample when it's selected and not uploaded. In case you wish to save the result, downoad the fonts with the licence, and then use the loader in the app.

Included samples:

- `facade-ouest.dfont` - [Facade Ouest by Eleonore Fines](https://velvetyne.fr/fonts/facade/), Velvetyne Type Foundry; licensed under the SIL Open Font License 1.1.
- `trickster-regular.dfont` - [Trickster by Jean-Baptiste Morizot](https://velvetyne.fr/fonts/trickster/); licensed under the SIL Open Font License 1.1.
- `picnic-regular.dfont` - [PicNic by Mariel Nils](https://velvetyne.fr/fonts/picnic/); licensed under CUTE 0.1.

License and source-context files are bundled under `licenses/`.
