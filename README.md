# Glitch.dfont

Glitch.dfont is a browser-based `.dfont` bitmap font glitching app and lab.

Public app: https://chudotony.github.io/glitch-dfont/

The app runs entirely in the browser. Uploaded or dropped files are processed locally in the page and are not sent to a server.

## Use

Select an included sample, or upload/drop a `.dfont` file, then adjust the glitch controls and download the resulting font file.

Bundled samples live in `docs/samples`:

- Facade Ouest by Eleonore Fines / Velvetyne Type Foundry, SIL Open Font License 1.1
- Trickster by Jean-Baptiste Morizot, SIL Open Font License 1.1
- PicNic by Mariel Nils, CUTE 0.1

`docs/samples/licenses/` contains the bundled license and source-context files for these samples.

## Repository Structure

- `docs/` - GitHub Pages app
- `docs/samples/` - hosted samples
- `research/` - original coursework and process archive

The project began as a Digital Culture class experiment and became a standalone browser tool.
